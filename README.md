# 🤖 AI Career Advisor

An AI-powered Career Advisor built using **Python, Flask, and Google's Gemini API**. It analyzes a user's current skills and career goals to generate a personalized roadmap, skill gap analysis, project ideas, resume tips, and interview preparation guidance.

---

## 📌 Features

- 🎯 Personalized Career Roadmap
- 📊 Skill Gap Analysis
- 📚 Learning Recommendations
- 💼 Portfolio Project Suggestions
- 📄 Resume Improvement Tips
- 🎤 Interview Preparation Guidance
- 🤖 Powered by Google Gemini AI

---

## 🛠 Technologies Used

- Python
- Flask
- Google Gemini API
- HTML
- Tailwind CSS
- python-dotenv

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/OmDangmali2103/ai-career-advisor.git
```

Move into the project folder

```bash
cd ai-career-advisor
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
